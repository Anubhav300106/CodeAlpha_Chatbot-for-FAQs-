About This Project

AI FAQ Chatbot answers frequently asked questions using Natural Language Processing. This live demo preprocesses text (tokenize, remove stopwords, stem), converts FAQs into TF-IDF vectors, and finds the best match using cosine similarity — all running directly in your browser.

  • 🧠 NLP: TF-IDF + Cosine Similarity (vanilla JS implementation)
  • ⚙️ Backend equivalent: Python Flask + NLTK + scikit-learn version included in your download
  • 🎨 Frontend: HTML5, CSS3 (Glassmorphism), Vanilla JS
  • 📚 Dataset: 60 curated FAQs across 16 categories

How to Use

  1. Type your question in the input box, or tap a suggested/example question.
  2. Press Enter or click Send to get an instant answer.
  3. Use the sidebar to jump to topics like Admission, Library, or Fees.
  4. Toggle Dark Mode from the top-right corner anytime.
  5. Click Clear Chat or New Chat to start fresh.
  6. Use Export Chat to download your conversation as a text file.
